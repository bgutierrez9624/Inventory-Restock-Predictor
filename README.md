# Inventory-Restock-Predictor

This program will allow users to input multiple item name, current stock, and average daily sales to determine if you should restock now or wait another week.   

# Purpose

writing a program that can tell me if an item needs restocking or not. This program could potentially alleviate the stress and time consumption that ordering inventory can create.

# Features

•Input multiple item names 

•Input current stock available 

•Input average daily sales 

•Output whether or not you need to restock and how much of an item you should buy to maintain your inventory for 7 days

# Future Features

•Use data from a CSV file to automatically update inventory requirments
